# cli

An index of my command-line tools.

This repo mirrors the role of [`mcps`](https://github.com/adinschmidt/mcps), but for standalone CLIs instead of MCP servers.

| Package | Status | Description |
|---------|--------|-------------|
| `fastmail-cli` | local-only | Planned CLI for Fastmail mail (JMAP), calendar (CalDAV), and contacts (CardDAV) |
| [splitwise-cli](https://github.com/adinschmidt/splitwise-cli) | published | Production-quality Rust CLI for the Splitwise API |
